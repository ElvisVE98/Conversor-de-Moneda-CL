FinChile – Conversor de divisas en tiempo real

Aplicación web desarrollada como proyecto personal para practicar desarrollo frontend utilizando TypeScript, JavaScript, HTML y CSS.

El proyecto consiste en un conversor de divisas enfocado en el mercado chileno, que permite convertir entre CLP, USD, EUR y UF utilizando datos actualizados obtenidos desde la API pública de MiIndicador.cl.

Además del conversor, la aplicación muestra indicadores económicos relevantes como Dólar, Euro, UF y UTM, junto con su variación y valores actualizados.

La interfaz fue diseñada con un enfoque moderno utilizando modo oscuro, efectos visuales, animaciones y diseño responsivo, buscando simular una experiencia similar a dashboards financieros actuales.
